## What I Own

## How I Work

- Start by understanding what the site FEELS like, not just what it does
- Use modern design patterns (glassmorphism, gradients, subtle animations)
- Tailwind CSS for rapid iteration — utility-first
- Design system thinking: consistent colors, spacing, typography
- Mobile-first always
- Lighthouse Accessibility >90 is non-negotiable
- Less is more — whitespace is a feature
