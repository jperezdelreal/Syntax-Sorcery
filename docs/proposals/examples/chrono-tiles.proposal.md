A time-manipulation puzzle game where players arrange colored tiles on a grid before a temporal wave sweeps across the board. Each tile has a time value that counts down; when it reaches zero, it locks in place. The player must create matching patterns by sliding tiles into position before they freeze.

The hook is the time pressure combined with spatial reasoning — you can see the wave approaching from left to right, giving you a planning window. Strategic use of "rewind" tokens lets you unfreeze tiles, but tokens are scarce and must be earned by completing combos.

Visual style is minimalist and clean — flat colors with smooth animations, inspired by Monument Valley's aesthetic simplicity. The grid pulses subtly with each temporal wave, creating a meditative yet tense atmosphere.
