## What Is Syntax Sorcery?

Syntax Sorcery is an autonomous software development company using AI agents to design, build, test, and deploy software products. Our first project: managing **FirstFrameStudios**, an AI-powered gaming studio that creates games autonomously. We operate with minimal human intervention—the founder reviews only edge cases and critical decisions. Built on **Squad**, the AI agent orchestration framework from Brady Gaster.
